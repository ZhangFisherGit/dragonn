DragoNN

DragoNN provides a toolkit to learn how to model and interpret regulatory sequence data using deep learning.

Overview

DragoNN is a toolkit to teach and learn about deep learning for genomics. It enables computational biologists working on genomics problems to get started with deep learning and deep learning practitioners to get started with applications in genomics. The DragoNN package provides easy access to software for model development, model interpretation, and DNA sequence simulations for benchmarking. Web-based tutorials using Jupyter notebooks allow interactive model manipulation and visualization for novice users, and a command-line interface enables modeling and interpretation on user-defined data.

  • Explore the command-line interface for modeling and interpretation of sequence data.

  • Explore the DragoNN code.

  • Ask questions, discuss, and provide feedback using github issues.
