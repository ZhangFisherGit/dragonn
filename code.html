DragoNN

DragoNN provides a toolkit to learn how to model and interpret regulatory sequence data using deep learning.

Overview Tutorials Code Documentation Workshops Paper Supplement

Installing the latest released DragoNN version

Install python Anaconda. Once you have installed Anaconda for your computing platform, the latest released version of DragoNN can be installed with the following command from the terminal:

conda install -c kundajelab dragonn

Installing the bleeding-edge DragoNN version

First, install the latest released DragoNN version (see above). Then, clone the DragoNN repository and run:

python setup.py install

Tutorials can be found in ./examples.
